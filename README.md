This is the the private events project from The Odin Project
https://www.theodinproject.com/lessons/ruby-on-rails-private-events.
The purpose is to create a website that mimics the functions of Eventbrite (https://www.eventbrite.com/) Where users may view other events and create their own.
In this project, the events created may be public/private, decided by the creator. Events can only be joined by users if the creator invites that particular user.

This is my second attempt at this project. My first attempt became a big mess, so I decided to start over from scratch.
Lessons learnt: How to setup has_many :through associations between two models with a many_to_many relationship. Further practice into creating forms, using radiobuttons, using enums, scopes.
Lastly, using proper authentication and authorization through the devise gem.
